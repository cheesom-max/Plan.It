# Product Requirements Document (PRD)
# AI Travel Planner (Plan.It)

**버전**: 1.0.0  
**최종 수정일**: 2026-01-14  
**상태**: Released

---

## 1. 제품 개요

### 1.1 제품 비전
AI Travel Planner는 사용자가 여행 계획에 소요하는 시간과 노력을 최소화하면서, AI를 통해 개인 맞춤형 최적의 여행 일정을 제공하는 웹 서비스입니다.

### 1.2 목표 사용자
- 여행 계획 초보자
- 시간이 부족한 직장인
- 다중 목적지 여행 계획이 필요한 여행자
- 새로운 여행지 탐색을 원하는 사용자

### 1.3 핵심 가치
- **편의성**: 복잡한 여행 계획을 AI가 자동화
- **개인화**: 사용자 취향에 맞는 맞춤형 일정
- **글로벌**: 전 세계 모든 도시 지원

## 2. 기능 요구사항

### 2.1 도시 검색 (P0 - 필수)

| 기능 | 설명 | 상태 |
|------|------|------|
| 글로벌 검색 | OpenStreetMap API로 전 세계 도시 검색 | ✅ 완료 |
| 자동완성 | 타이핑 시 실시간 검색 결과 표시 | ✅ 완료 |
| 다중 목적지 | 여러 여행지 추가/삭제 | ✅ 완료 |

### 2.2 여행 정보 입력 (P0 - 필수)

| 기능 | 설명 | 상태 |
|------|------|------|
| 날짜 선택 | 출발일/도착일 캘린더 | ✅ 완료 |
| 동행자 선택 | 혼자/친구/연인/가족 라디오 버튼 | ✅ 완료 |
| 여행 스타일 | 8종 스타일 다중 선택 태그 | ✅ 완료 |

### 2.3 AI 일정 생성 (P0 - 필수)

| 기능 | 설명 | 상태 |
|------|------|------|
| 일정 생성 | Gemini AI 기반 상세 일정 생성 | ✅ 완료 |
| 일별 분류 | 날짜별 타임라인 형태 표시 | ✅ 완료 |
| 활동 상세 | 시간, 장소, 설명, 유형 포함 | ✅ 완료 |
| 여행 팁 | AI 추천 여행 팁 제공 | ✅ 완료 |

### 2.4 일정 관리 (P1 - 중요)

| 기능 | 설명 | 상태 |
|------|------|------|
| 로컬 저장 | localStorage 기반 일정 저장 | ✅ 완료 |
| 인쇄 기능 | 일정 프린트 출력 | ✅ 완료 |
| 새 일정 생성 | 메인 화면으로 돌아가기 | ✅ 완료 |

## 3. 비기능 요구사항

### 3.1 성능
- 도시 검색 응답: < 1초
- AI 일정 생성: < 15초
- 페이지 로드: < 3초

### 3.2 호환성
- 최신 Chrome, Firefox, Safari, Edge 지원
- 반응형 디자인 (모바일, 태블릿, 데스크탑)

### 3.3 보안
- API 키 서버사이드 보관
- HTTPS 통신

## 5. 화면 구성

### 5.1 메인 페이지 (index.html)
- 히어로 섹션 (그라디언트 배경, 슬로건)
- 여행 정보 입력 폼
  - 목적지 검색 (다중 입력)
  - 날짜 선택
  - 동행자 선택
  - 여행 스타일 선택
- CTA 버튼 (일정 생성하기)

### 5.2 일정 결과 페이지 (itinerary.html)
- 여행 제목 및 요약
- 일별 타임라인 뷰
- 활동 카드 (시간, 장소, 설명)
- 여행 팁 섹션
- 액션 버튼 (저장, 인쇄, 새 일정)
