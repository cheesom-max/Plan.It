# ✈️ AI Travel Planner

AI 기반 맞춤형 여행 일정 생성기 웹 서비스

![Hero Section](images/hero-bg.png)

## 🌟 프로젝트 소개

AI Travel Planner는 사용자의 여행 정보를 입력받아 인공지능이 최적의 여행 일정을 설계해주는 웹 서비스입니다.

## 🎯 주요 기능

- **📍 도시 검색**: 자동완성 기능으로 20개 이상의 인기 여행지 검색
- **📅 날짜 선택**: 출발일/도착일 캘린더
- **👥 동행자 선택**: 혼자, 친구, 연인, 가족 중 선택
- **🎨 여행 스타일**: 맛집, 휴양, 액티비티, 문화예술 등 복수 선택
- **✨ AI 일정 생성**: 입력 정보 기반 맞춤 일정 자동 생성

## 🛠️ 기술 스택

- HTML5
- CSS3 (애니메이션, 그라디언트, 글래스모피즘)
- Vanilla JavaScript
